# caplos

A tool to squeeze the power of your CapsLock key

一个充分利用你的大小写键的工具, 可以最小化打字/写代码时的按键距离.

## 功能

* 大小写状态指示
* 按键替换

## 使用方法

按下\[大小写键\]的同时按下其它按键, 可执行替换后的按键的功能.

举例:

比如你想不移动右手位置就输入上下左右, 你可以使用组合键 `[大小写]+H/J/K/L`

比如删除键太远了你不想去用小拇指够, 你可以使用组合键 `[大小写]+'`
